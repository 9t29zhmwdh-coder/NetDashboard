# Privacy Policy

NetDashboard is a local Windows desktop application. All network diagnostic data,
mail server test results, and configuration remain on your machine.

- **No telemetry:** The application does not send usage data to any external server.
- **No cloud sync:** All data stays local.
- **Mail credentials:** Any SMTP/IMAP credentials you enter are stored locally in Windows credential manager or config files — never transmitted externally by the app.

Network traffic generated by the diagnostic tools goes only to the hosts you specify.
