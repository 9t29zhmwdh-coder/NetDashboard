# NetDashboard

**Netzwerk- & Mail-Toolkit** — by [RayStudio](https://github.com/9t29zhmwdh-coder)

![NetDashboard Screenshot](screenshot.png)

## Features

- **Mail-Check** — E-Mail-Adresse eingeben und IMAP/SMTP-Einstellungen automatisch ermitteln
- **DNS-Check** — DNS-Abfragen über Google, Cloudflare und Quad9 parallel vergleichen
- **Verbindung** — Netzwerkverbindung prüfen und diagnostizieren
- **Mein System** — Systeminformationen auf einen Blick

## Voraussetzungen

- Windows 10/11
- .NET 8.0 Runtime

## Technologien

- C# / WPF (.NET 8)
- Dark-Theme UI
